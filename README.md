Bobcat Proxy — это прокси-клиент с самой свежей стабильной версией xray-core а также naive proxy. Его основная задача — решить проблему Happ, который обновляет xray-core с огромной задержкой, что позволяет РКН быстрее выявлять сервера и эксплуатировать уязвимости. Отдельный репозиторий версии с Naive Proxy в профиле. UPD: Naive Proxy пока только под Linux
